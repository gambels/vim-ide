# vim-ide

## Install vim-ide
```bash
git clone --recurse-submodules https://github.com/beckenc/vim-ide.git
```

### .. for current user
```bash
./vim-ide/install.sh
```

### .. for dedicated users
```bash
./vim-ide/install.sh <usr1> <usr2>
```

### .. for all users
```bash
./vim-ide/install.sh --all
```

## Plugin Manager

### Install Plugins
```vim
:PackUpdate
:helptags ALL
```

### Clean Plugins
```vim
:PackClean
```
